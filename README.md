# CC-LancertionMLAPI-old-model

## Usage

- Clone repo

```
git clone https://github.com/Lancertion/CC-LancertionMLAPI-new-model.git
cd CC-LancertionMLAPI-new-model
```

- Run app.py

```
python app.py
```

- Send POST request to http://localhost:3000

Body request as JSON:

```
{
    "alcohol_use": 4,
    "dust_allergy": 5,
    "genetic_risk": 7,
    "balanced_diet": 6,
    "obesity": 6,
    "smoker": 4,
    "coughing_of_blood": 7
}
```
